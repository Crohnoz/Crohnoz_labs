# Cronhoz Labs V6

## Objetivo
Refinar la experiencia para que el sitio sea más profesional, menos largo y más fácil de explorar.

## Cambios principales

- La portada quedó compacta y orientada a conversión.
- Se creó una nueva ruta `/explorar` para mover contenido extendido fuera del home.
- Se reorganizó el navbar en navegación principal y menú secundario.
- Se agregó `BrandSystem.jsx` para ordenar la narrativa de marca y familias de solución.
- Se creó `brand.js` para centralizar principios de marca y arquitectura de soluciones.
- Las tarjetas de proyectos ahora son más limpias y comerciales.
- Las muestras públicas siguen separadas del producto real del cliente.
- Se agregó `/explorar` al sitemap.

## Home V6

La portada ahora contiene solo:

1. Hero
2. Lectura rápida
3. Brand system
4. Soluciones
5. Proyectos
6. Método
7. CTA final

## Contenido extendido

La ruta `/explorar` contiene:

- Problemas

- Infraestructura
