# Cronhoz Labs V8 — Branding C_ aplicado

Esta versión aplica la dirección de marca aprobada: **C_** como símbolo principal inspirado en terminal, comando activo, infraestructura real y sistemas que funcionan.

## Cambios principales

- Nuevo símbolo oficial `C_` integrado en `BrandMark.jsx`.
- Nuevo favicon SVG con isotipo `C_`.
- Nuevo logo horizontal SVG en `public/logo-cronhozlabs.svg`.
- Nueva imagen Open Graph en `public/og-cronhozlabs.svg`.
- Paleta ajustada hacia verde terminal + azul datos:
  - Obsidiana operativa `#0B0F14`
  - Panel grafito `#111827`
  - Verde terminal `#10B981`
  - Azul datos `#38BDF6`
  - Blanco frío `#F2F4F7`
- Claim actualizado: **Sistemas que hacen funcionar tu negocio.**
- Hero reescrito para comunicar automatización, infraestructura y soporte real.
- Tailwind ajustado para usar Space Grotesk como tipografía principal.
- Metadata SEO/OG actualizada al nuevo posicionamiento.
